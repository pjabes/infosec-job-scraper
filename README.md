# InfoSec Job Scraper

## Description

Python scripts to automate the scraping of jobs for career analysis.

## Getting Started

### Dependencies

* Python3 
* Libraries
* Internet Connection

### Installing

``` 
git clone https://github.com/pjabes/infosec-job-scraper.git
pip install -r requirements.txt
touch jobs.csv
```

### Executing program
```
python3 scrapper.py
```
